# Copilot Prompt Library (PHP CodeIgniter + ReactJS)

Reusable Copilot Chat prompts to quickly invoke the right skill/model.

---

## 🐘 PHP CodeIgniter Prompts
- `@project-assistant generate a CodeIgniter controller for user authentication`
- `@project-assistant create a migration for adding a comments table`
- `@project-assistant generate PHPUnit tests for this controller`

## ⚛️ ReactJS Prompts
- `@project-assistant create a functional React component with hooks for a login form`
- `@project-assistant implement Redux slice for user authentication`
- `@project-assistant generate Jest tests for this component`

## 🧠 Deep Reasoning (Claude Sonnet 4.6)
- `@project-assistant refactor the auth service for scalability`
- `@project-assistant design a scalable API architecture for CodeIgniter`

## 📄 Documentation
- `@project-assistant generate README for this project`
- `@project-assistant create API documentation for these endpoints`
